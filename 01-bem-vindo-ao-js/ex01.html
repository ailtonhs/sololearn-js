Indrodução JavaScript

Você pode adicionar código JavaScript diretamente nas tags HTML. O exemplo abaixo usa o atributo onclick para exibir uma mensagem de alerta

O onclick faz uma ação acontecer quando o usuário clica em um elemento.

A forma mais simples de interatividade consiste em um evento iniciado pelo usuário que aciona uma ação.

Para exibir um alerta, inclua a mensagem dentro dos parênteses().

        
            <button onclick="alert('mensagem')">Enviar</button>
