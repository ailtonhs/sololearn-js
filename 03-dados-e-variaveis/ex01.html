Dados e Variáveis

Strings JavaScript

Um pedaço de dados de texto é chamado de strin. Você pode usar aspas simples ou dupas para criar uma string.

As aspas garantem que o pedaço de dados seja entendido e tratado como texto.

        
            console.log("Olá, Mundo!");

Números JavaScript

Números no código JS não devem ser colocados entre aspas.

        
            <script>
                alert(500);
            </script>

Variáveis JavaScript

Os programas de computador usam Variáveis para armazenar informações importantes, como itens em um carrinho de compras, preços e descontos.

Para criar variável, use a palavra-chave let seguida pelo nome da variável.

Uma variável tem um name e um value.

Para criar um nome, dê um nome à variável e conecte o nome e o valor com um igual = sinal.

        
            let nome = "Ailton";
