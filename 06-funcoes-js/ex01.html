Funções JavaScript

Funções são blocos de construção chave no JavaScript. Eles vão de ajudar a tornar suas páginas web interativas e dinâmicas.

Declarações JavaScript

Declarações pode ser agrupadas em blocos de código. Chave{} são usadas para criar um bloco de código

                
                    {
                        let x = "Game Over";
                        alert(x);
                        console.log(x);
                    }

Funções JavaScript

Uma função é um bloco de código que executa uma tarefa específica. Use a palavra-chave function seguida por um name para criar uma nova função.

                
                    function doSomething() {
                        let x = "Game Over";
                        alert(x);
                        console.log(x);
                    }

Chamando uma Função

Uma função contém o código para executar uma tarefa. Este código será executado quando a função for Chamada. Para chamar uma função use seu nome seguido de parênteses().

Vai acontecer alteração nesse parágrafo quando a função for chamada.
