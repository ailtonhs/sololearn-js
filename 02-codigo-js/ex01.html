JavaScript Onde adicionar

JavaScript Inline

O código JS pode ser adicionado diretamente em elementos HTML

            
                <button onclick="alert('Olá, Mundo!')"

JavaScript Interno

Todo o código JS para uma página web pode ser agrupado dentro do documento HTML dentro da tag de container <script>. Esse método é conhecido com JavaScript Interno. 

            
                <script>
                    alert("Olá Mundo!");
                </script>

Navegadores web leem e executam instruções JavaScript. Cada instrução ´chamada de declaração. Você pode incluir quantas declarações precisar em seu script.
